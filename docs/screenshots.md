# Screenshots Guide

This folder is reserved for screenshots and demo images of cs-visualizer.

Screenshots are useful for:

- the README
- GitHub repository preview
- portfolio website cards
- LinkedIn project posts
- future case studies

## Recommended screenshots

Capture one screenshot for each main section:

- Home or main app layout
- Sorting visualizer
- Graph traversal visualizer
- Heap visualizer
- Linked list visualizer
- State machine visualizer

## Suggested filenames

Use clear lowercase filenames:

- home.png
- sorting-visualizer.png
- graph-traversal.png
- heap-visualizer.png
- linked-list-visualizer.png
- state-machine.png

## Screenshot tips

For the best presentation:

- use a clean browser window
- avoid showing bookmarks or personal tabs
- use desktop width for main README screenshots
- capture the visualizer while an animation step is highlighted
- show controls and explanation panels when possible

## README usage

Once screenshots are added, they can be embedded in the README using normal Markdown image links.

Example:

![Sorting visualizer](public/screenshots/sorting-visualizer.png)

## Current status

The screenshots folder is intentionally included with a .gitkeep file so the directory exists before images are added.
