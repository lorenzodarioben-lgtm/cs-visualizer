# Roadmap

This roadmap lists future improvements for `cs-visualizer`.

The current project already includes working visualizers for sorting, graph traversal, heaps, linked lists, and a finite state machine. These ideas would make the project larger, more interactive, and more useful as a computer science learning tool.

## Algorithm visualizers

Planned visualizers:

- Dijkstra shortest path
- A* pathfinding
- Binary search
- Binary search tree operations
- AVL tree rotations
- Hash table insertion and collision handling
- Stack and queue operations
- Recursion call stack visualizer
- Dynamic programming table visualizer

## Sorting improvements

Future sorting upgrades:

- Add shell sort
- Add radix sort
- Add counting sort
- Add custom user input arrays
- Add comparison count and swap count
- Add best, average, and worst-case examples

## Graph improvements

Future graph upgrades:

- Custom graph editor
- Weighted edges
- Directed graph mode
- Dijkstra visualizer
- Cycle detection
- Topological sort
- Minimum spanning tree visualizer

## Heap improvements

Future heap upgrades:

- Max-heap mode
- Priority queue examples
- Heap sort visualizer
- Custom heap array input
- More detailed bubble-up and bubble-down explanations

## Linked list improvements

Future linked list upgrades:

- Doubly linked list mode
- Circular linked list mode
- Detect cycle demo
- Merge two sorted lists
- Find middle node
- Remove nth node from end

## State machine improvements

Future state machine upgrades:

- Custom transition table editor
- More example machines
- Input validation examples
- Regex-style automata demos
- Export transition history

## UX improvements

Future UI polish:

- Dark and light mode toggle
- Better mobile layouts
- Animation timeline scrubber
- Keyboard shortcuts
- Screenshot capture button
- Step export as JSON
- More accessible color themes

## Documentation improvements

Future documentation:

- Add screenshots
- Add demo GIFs
- Add contribution guide
- Add deployment guide
- Add learning notes for each data structure
- Add diagrams explaining the step-generation architecture

## Deployment ideas

Possible hosting options:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

## Long-term goal

The long-term goal is to make this project feel like a polished computer science learning lab rather than a small demo.

Each visualizer should be interactive, educational, tested, and easy to understand from both a learner perspective and a developer perspective.
